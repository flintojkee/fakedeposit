# AngularPrerenderStarter

Angular Prerender Starter Project 

## Prerequisites

```
npm i local-web-server -g
```

## Installation

```
git clone https://github.com/pikselinweb/fakedeposit.git
cd fakedeposit
npm i
```

## Build

```
npm run build:all
```
## Serve
```
npm run serve:prerender
```
## Deploy To Firebase

```
firebase init hosting
```
